# 🚀 AI & ML Mastery Journey

> Building deep, practical machine learning expertise — from Python fundamentals to deploying production AI systems.

## 👋 About This Repository

This repo documents my hands-on journey learning and refreshing machine learning and AI, structured as a self-paced curriculum I'm working through phase by phase. The goal isn't just to learn ML, it's to build the depth and confidence to lead an AI-driven business, evaluate technical decisions critically, and eventually hire and manage a world-class ML team.

Each phase lives in its own folder with notebooks, exercises, and a README explaining what was covered.

## 📊 Highlights So Far

- Built and compared 7 classic ML algorithms on the Titanic dataset, with **XGBoost** as the top performer at **77% accuracy**
- Trained a neural network (`PatientNet`) from scratch in PyTorch, dropping training loss from **5.96 → 0.26** over 100 epochs
- Used transfer learning with a pretrained ResNet18, training just **1,026 of 11+ million parameters (0.01%)**
- Directly observed overfitting vs. generalization by comparing training curves with and without dropout
- Built a reusable custom Python module (`ai_utils.py`) and an inheritance-based class hierarchy for AI project reporting
